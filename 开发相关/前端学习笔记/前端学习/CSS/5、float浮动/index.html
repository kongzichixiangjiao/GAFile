Float浮动的基本知识

  • 会使元素向左或向右移动，只能左右，不能上下
  • 浮动元素碰到包含框或另一个浮动框，浮动停止
  • 浮动元素之后的元素将围绕它，之前的不受影响
  • 浮动元素会脱离标准流

示例

img浮动，这段话在浮动之前，不受影响。

清除浮动

  • /* 第一种清除浮动的方法 */ /* .clear { clear:both; } */ /* 后面需要清除浮动的标签添加 */
  • /* overflow: hidden; */ /* 父标签添加。第二种清除浮动的方法*/
  • /* 清除浮动的第三种方法 */ .clearfix:after { content: '.'; height: 0; display: block; visibility: hidden; clear: both; } zoom: 1; /* 清除浮动的第三种方法 适配低版本IE*/
